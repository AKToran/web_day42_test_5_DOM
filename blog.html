Question-3: What is event delegation in the DOM?

Event delegation is to write an event listener on the parent element instead of all the child elements. It takes advantage of event bubbling.

Question-4: What is event bubbling in the DOM?

When an event takes place in a child element it bubbles upto its parent element, gradually bubbling upto all the parent elements up to the root element. First it runs the event handlers where the event occurred, then runs the event handlers of its parent, then their parent and so on.

Question-5: How do you create, add, and remove elements using JavaScript?

Create element: document.createElement('tag-name')
Add element: appendChild method or append method. First select the element where we want to add a new element. Second, create the element we want to add and write the inner html. Then use appendChild method.
Remove element: Using remove method: document.getElementById('id').remove()
